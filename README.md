# HeartBank® publications

The institutional-output corpus for [heartbank.net](https://heartbank.net) — HeartBank®'s institutional publications across four document genres.

## Genres

| Genre | Subdirectory | Description |
|---|---|---|
| **White Papers** | [`white-papers/`](./white-papers/) | 8–15 pages, with embedded executive summary; synthesizes across multiple research papers; addresses policy/strategy questions. Slow cadence (1–2/year). |
| **Position Papers** | [`positions/`](./positions/) | 1–3 pages; institutional stance on a specific topic. |
| **Annual Reports** | [`annual-reports/`](./annual-reports/) | Year-end institutional reporting; corpus progress, governance updates, milestone summary. First report covers 2026; published early 2027. |
| **Press Releases** | [`press/`](./press/) | Event-driven announcements; new paper publications, partnerships, deployments. |

## Voice

Institutional voice throughout. Third-person institutional. Mechanism-led. Sober register. Author: HeartBank® (Founder: Thon Ly; CEO: Miss Aquarius, the project's named AI substrate, `miss.aquarius@heartbank.ceo`).

## Paired with

This repo is paired with [`github.com/thonly/publications`](https://github.com/thonly/publications) — the author-output corpus for thonly.org (defensive publications, essays, treatments, screenplays; founder voice).

The two repos publish *different document genres*, not the same content in two voices. HeartBank® speaks as institution; Thon Ly speaks as author. Cross-link honestly.

## Relationship to heartbank.net deployed Lit pages

Each markdown here corresponds to a deployed page at heartbank.net:

- `white-papers/{slug}.md` ↔ `heartbank.net/white-papers/{slug}`
- `positions/{slug}.md` ↔ `heartbank.net/positions/{slug}`
- `annual-reports/{year}.md` ↔ `heartbank.net/annual-reports/{year}`
- `press/{slug}.md` ↔ `heartbank.net/press/{slug}`
